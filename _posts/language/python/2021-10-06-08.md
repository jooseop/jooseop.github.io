# Heapq  

* 우선순위 큐 대신 사용 가능 
* 맨 첫번째 값은 최솟값 
  * heapify 
    * 기존에 있는 리스트를 힙으로 변경시켜준다. 
    * 빈 리스트를 사용한다면, 그냥 heappush, heappop 쓰면됨



# Import  
```python  
from heapq import heappop, heappush, heapify, heappushpop, heapreplace
```
